## Customizable Experience
Litlyx supports `Custom Events` for a fully customized experience, giving you complete control over what user behavior to track. You decide which actions to monitor. Whether you're tracking page visits or custom events, Litlyx handles both seamlessly, with no hidden costs for using either.
